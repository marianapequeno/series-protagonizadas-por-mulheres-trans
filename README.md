# Séries protagonizadas por mulheres trans e travestis

### Sobre o projeto:
* Página responsiva feita **HTML** e **CSS**; 
* Populada com os dados do arquivo **.JSON**, simulando consumo de uma API;
* Os dados estão no arquivo *data.json*.
